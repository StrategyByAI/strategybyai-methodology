# Document Portfolio Structure

## Complete 34-Document Reference

### Folder One: Strategic Identity Portfolio
*From Module Two: Strategic Mission, Strategic History, Strategic Doctrine*

| # | Document | Function |
|---|----------|----------|
| 1 | Strategic Mission Intelligence Report | Vision, conditions, interventions, vocation. Ultimate reference for strategic decisions. |
| 2 | Strategic History Intelligence Report | Historical trajectory, past conflicts, capability development. Organisational strengths/weaknesses in action. |
| 3 | Strategic Doctrine Intelligence Report | Principles, approaches, ways/means, modes/timelines. Decision-making under uncertainty. |
| 4 | Strategic Identity Intelligence Profile | Mission-history-doctrine integration. Identity-threat correspondence. |

### Folder Two: Hostile Environment Portfolio
*From Module Three: Enemy-Making*

| # | Document | Function |
|---|----------|----------|
| 5 | Hostile Environment Intelligence Assessment | Complete environmental opposition mapping. Opposition force characterisation. |
| 6 | Foe Status Intelligence Matrix | Adversary hierarchy and prioritisation. Resource allocation priorities. |
| 7 | Confrontation Hierarchy Map | Primary/secondary/tertiary confrontation structure. Multi-front management. |
| 8 | Resource Allocation Audit | Current resource distribution and adequacy. Culmination point prediction. |
| 9 | Enemy Strategic Profiles | Detailed adversary capabilities and patterns. Opposition exploitation assessment. |
| 10 | Conflict Characterisation Matrix | Conflict nature and intensity specification. Tempo and friction requirements. |
| 11 | Outcome Scenarios Document | Victory/defeat/stalemate conditions. Settlement orientation. |
| 12 | Multi-Front Confrontation Strategy | Simultaneous confrontation management. Campaign coordination. |
| 13 | Threat Monitoring and Early Warning System | Active threat identification. Real-time opposition updates. |

### Folder Four: Strategy Formulation Portfolio
*From Module Five: Making the Strategy*

| # | Document | Function |
|---|----------|----------|

### Folder Five: Strategy Execution Portfolio
*From Module Six: Execution of the Strategy*

| # | Document | Function |
|---|----------|----------|
